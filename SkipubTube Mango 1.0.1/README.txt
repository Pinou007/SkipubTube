FRANCAIS ------------------------------------------------------- - -

SkipubTube est un projet open source visant à éliminer les publicités sur la plateforme YouTube et à passer outre les vidéos publicitaires. Ce projet a pour vocation de vous offrir une solution pratique, mais il est impératif qu'il demeure la propriété exclusive de son créateur. En aucun cas ce projet ne doit être diffusé sur une quelconque plateforme d'extension ni quitter l'écosystème de GitHub.

Vous êtes autorisé à apporter des modifications au code et à en faire part au développeur, Pinou007. Cependant, il est crucial de garder à l'esprit que les publicités sont une source de revenus cruciale pour les créateurs de contenu. Avant d'utiliser un bloqueur de publicités, veuillez réfléchir à l'impact que cela peut avoir sur ces créateurs.

Il est important de noter que Pinou007 ne peut en aucun cas être tenu responsable de l'utilisation de cette extension ni des éventuelles sanctions qui pourraient vous être infligées. Ce projet est exclusivement à vocation éducative. Merci de votre compréhension.

ENGLISH --------------------------------------------------------- - -

SkipubTube is an open-source project with the aim of eliminating advertisements on the YouTube platform and skipping advertising videos. This project is designed to provide you with a practical solution, but it is essential that it remains the exclusive property of its creator. Under no circumstances should this project be distributed on any extension platform or depart from the GitHub project repository.

You are allowed to make modifications to the code and communicate these changes to the developer, Pinou007. However, it is crucial to bear in mind that advertisements are a vital source of revenue for content creators. Before using an ad blocker, please consider the potential impact on these creators.

It's important to note that Pinou007 cannot be held responsible for the use of this extension or any potential sanctions that may be imposed on you. This project is solely for educational purposes. Thank you for your understanding.
